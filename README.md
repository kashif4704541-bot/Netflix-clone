# Netflix Clone 🎬

A simple Netflix homepage clone built using only **HTML** and **CSS**.

## 🔥 Features

- Responsive design and Styled like the real Netflix landing page
- Hover effects and layout similar to original
- Pure HTML + CSS, no JavaScript
